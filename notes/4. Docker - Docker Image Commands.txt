To launch all the containers like for java, python, and nodejs at the same time
with different host port in same cmd window

In cmd or terminal : (-d = --detached)
    * docker run -d -p 5000:5000 in28min/hello-world-nodejs:0.0.1.RELEASE
    * docker run -d -p 5001:5000 in28min/hello-world-python:0.0.1.RELEASE
    * docker run -d -p 5002:5000 in28min/hello-world-java:0.0.1.RELEASE
    * docker logs (id - which is generated by hitting above command or 
			we can also enter first 4 characters of id )
    * docker logs -f (id -> -f is used to connect to the particular log )
   
To check like what are the images which are present in the local machine, and 
all the stuff

  In cmd hit : 
      * docker images
      * docker container ls ( to see the container list which are running 
				- ls is for list )

  to stop the particular container use command
      * docker container stop (id - which is generated by hitting above command or 
			we can also enter first 4 characters of id )
      * docker pull mysql ( pull the latest release form server )

      * docker search (images name - this cmd used for searching official imges)

      * docker images ( to see no of local images available )

      * docker image inspect (IMAGE ID - that you will get from docker images
			command. this will show info about image id information)

      * docker image remove (image name like mysql)

      * docker container rm (tab name or image id)
